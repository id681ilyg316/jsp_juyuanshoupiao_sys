## 本项目实现的最终作用是基于JSP剧院售票系统
## 分为2个角色
### 第1个角色为管理员角色，实现了如下功能：
 - 查看数据统计
 - 演出团队管理
 - 演出管理
 - 用户管理
 - 管理员登录
 - 订单管理
### 第2个角色为用户角色，实现了如下功能：
 - 查看订单
 - 查询演出
 - 演出预定
 - 用户登录
 - 钱包管理
